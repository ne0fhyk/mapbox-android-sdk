public class safetranslatedpath
extends android.graphics.path
the safetranslatedpath class is designed to work in conjunction with safetranslatedcanvas to work around various android issues with large canvases. for the two classes to work together, call ondrawcyclestart(com.mapbox.mapboxsdk.views.safecanvas.isafecanvas) at the start of the com.mapbox.mapboxsdk.overlay.overlay.overlay#drawsafe method of your com.mapbox.mapboxsdk.overlay.overlay.overlay. this will set the adjustment needed to draw your path safely on the canvas without any drawing distortion at high zoom levels. methods of the path class that use unsafe float types have been deprecated in favor of replacement methods that use doubles.
